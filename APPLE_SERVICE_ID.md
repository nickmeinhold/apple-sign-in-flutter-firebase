Go to [Certificates, Identifiers & Profiles - Apple Developer > Service Ids](https://developer.apple.com/account/resources/identifiers/list/serviceId)

You may need to select Service Ids at the top right of the screen

<kbd><img width="600" src="https://user-images.githubusercontent.com/1059276/84144936-68e9e300-aa9c-11ea-9c4a-2f638954454d.png"></kbd>

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144947-6b4c3d00-aa9c-11ea-8164-8269e05cda77.png"></kbd>

Hit the + button 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144952-6d160080-aa9c-11ea-92cf-6721d5b952ff.png"></kbd>

Make sure Services IDs is selected and hit Continue 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144960-6edfc400-aa9c-11ea-88de-6b2724b3f90d.png"></kbd>

Enter a description and an Identifier 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144966-71421e00-aa9c-11ea-950a-16c1aee557e0.png"></kbd>

Hit Continue 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144973-73a47800-aa9c-11ea-9ed5-99040ed0be2c.png"></kbd>

Hit Register 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144980-7606d200-aa9c-11ea-86e9-9129cfc3fb89.png"></kbd>

Click on the new Service Id 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144986-77d09580-aa9c-11ea-8610-5508949aab57.png"></kbd>

Enable Sign In With Apple 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144988-7a32ef80-aa9c-11ea-93f1-91a944650e37.png"></kbd>

Click Configure 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84144994-7c954980-aa9c-11ea-812b-aff6e2e5d678.png"></kbd>

Select the App Id (that was created in the App Id step), and add 
- a domain (use anything, eg. the first part of the Return URL)
- a Return URL (created in the Server Setup step) 

Then Click Done 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84173761-10790c80-aac1-11ea-92f2-697c6a3bc094.png"></kbd>

Click Continue 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84145028-87e87500-aa9c-11ea-9a92-f682fce4615e.png"></kbd>

Click Save 

<kbd><img width="400" src="https://user-images.githubusercontent.com/1059276/84145036-8a4acf00-aa9c-11ea-8b39-f6a1bac7c565.png"></kbd>
